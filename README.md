我的 Emacs 配置
===============
##使用方式

1. 获取代码
> git clone https://github.com/daydayfree/my-dot-emacs.git my-dot-emacs

2. 更新本地配置
> my-dot-emacs/.emacs.d my-dot-emacs/.emacs

##使用说明

1. 支持
    + solarized-dark 主题
    + yaml-mode
    + rube-mode
    + highlight-chars
    + coffee-mode
    + sr-speedbar
